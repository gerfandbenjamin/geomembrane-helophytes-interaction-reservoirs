# Potential impacts of helophyte growth on PVC-P geomembranes in mountain reservoirs

## 📄 Description

This repository contains the data and R scripts used in the study:

**"Potential impacts of helophyte growth on PVC-P geomembranes in mountain reservoirs"**
Submitted to *International Journal of Geosynthetics and Ground Engineering (IJGGE)*.

The objective of this work is to assess the mechanical and chemical resistance of geomembranes exposed to helophyte growth under experimental conditions.

---

## 📁 Repository structure

```
├── data/
│   ├── Data_for_R/         # Dataset used in R scripts
│   ├── Processed_data/     # Cleaned datasets and other analyses
│   └── Raw_data/           # Raw datasets (as collected)
│
├── scripts/
│   ├── bg_0_analyses_Ligh Temp sensors.R
│   ├── bg_1_analyses_belowground direct measurements.R
│   ├── bg_2_fig & analyses_WinRHIZO measurements.R
│   ├── bg_3_fig & table & analyses_deg mechanical resistance.R
│   └── bg_4_fig & analyses_deg plasticizer content.R
```

---

## ▶️ How to run the analyses

1. Set your working directory
2. Update the `data_path` variable in each script
3. Run scripts 

---


## 🔁 Reproducibility

All analyses are fully reproducible from the provided datasets and scripts.
Exept for WinRHIZO analyses which need the software licence and all the original photographs (non provided in the raw data)
